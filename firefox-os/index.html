Што е Firefox OS?

Gonk – platform denomination for a combination of the Linux kernel and the HAL from Android Gecko – the web browser engine and application run-time services layer; Gaia – an HTML5 layer and user-interface system. но ова не е толку важно, важното е…
сѐ е HTML5, сѐ што сте виделе на web, сите библиотеки, сите стандарди се надоградува на постоечките API-ја за web страни, html5 проширен со нови можности можат да се користат и на Firefox за Android, и на Firefox за компјутер, стандардизирани преку w3c, што значи па и на други browser-и, Chrome подржува некои од API-јата ништо чудно и Opera да ги имплементира, бидејќи имаат присуство во мобилниот свет па дури и Microsoft кокетира со идејата
